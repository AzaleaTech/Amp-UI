# Record 录音

## 概述

录音组件。

## 示例

```html
基础用法

<a-record tip="长按发表意见"></a-record>
```

## API

### Props

| 参数       | 说明                  | 类型   | 默认值           | 必填 |
| ---------- | --------------------- | ------ | ---------------- | ---- |
| tip        | 底部提示性文本        | string | 长按录入详细信息 | 否   |
| icon-color | 图标颜色              | string | #3895F7          | 否   |
| icon-size  | 图标大小, 单位[`rpx`] | number | 160              | 否   |

### Events

| 事件名 | 说明              | 参数 |
| ------ | ----------------- | ---- |
| change | 长按录音/删除回调 | e    |
