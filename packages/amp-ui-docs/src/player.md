# Player 语音播放器

## 概述

用于语音播放。

## 示例

```html
基础用法

<a-player url="{{ audioUrl }}"></a-player>
```

## API

### Props

| 参数       | 说明                  | 类型   | 默认值  | 必填 |
| ---------- | --------------------- | ------ | ------- | ---- |
| a-class    | 自定义根节点样式类    | string | -       | 否   |
| url        | 音频链接              | string | -       | 是   |
| icon-color | 图标颜色              | string | #3895F7 | 否   |
| icon-size  | 图标大小, 单位[`rpx`] | number | 44      | 否   |

### Events

| 事件名 | 说明 | 参数 |
| ------ | ---- | ---- |
| 暂无   | -    | -    |
