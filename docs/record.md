# Record 录音

## 概述

录音组件。

## 示例

```html
基础用法
<a-record tips="长按发表意见" />
```

## API

### Props

| 参数 | 说明           | 类型   | 默认值             | 必填 |
| ---- | -------------- | ------ | ------------------ | ---- |
| tip  | 底部提示性文本 | string | '长按录入详细信息' | 否   |

### Events

| 事件名 | 说明              | 参数 |
| ------ | ----------------- | ---- |
| change | 长按录音/删除回调 | e    |
