# Player 语音播放器

## 概述

用于语音播放。

## 示例

```html
<a-player url="{{ audioUrl }}"></a-player>
```

## API

### Props

| 参数    | 说明              | 类型   | 默认值 | 必填 |
| ------- | ----------------- | ------ | ------ | ---- |
| a-class | 自定义 class 类名 | string | -      | 否   |
| url     | 音频链接          | string | -      | 是   |

### Events

| 事件名 | 说明 | 参数 |
| ------ | ---- | ---- |
| 暂无   | -    | -    |
